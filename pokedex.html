The PokeDex

This PokeDex shows all the Pokémon from generation 1 to 8. You can use the search function to find a Pokémon by name, element type or by region. Choosing a Pokémon will let you see more details about its evolutions and a set of interesting facts you can read.

Feature highlights

  • Created a "Network Engine" class to interact with the API call, using the Result type in the functions to make handling the response more readable. The "Network Engine" is used in a singleton pattern when used to make a data request from the API.

  • The UI of the app was made programmatically and focuses on the MVVM design pattern to keep UI and logic code separate.

  • The app implements a custom search controller that presents category choices laid out in a collection view that translates the chosen option as text to the search bar for filtering. The search controller also allows for searching by the Pokemon's name, type or region.
