Weather Wear

What is Weather Wear?

Weather Wear is a weather app that will give you suggestions on what to wear based on the items of clothing you catalog in the app. The app works by letting you select your location by dropping a pin on the map and getting the 7-day outlook for that location. You can then select a day of the week and see what clothing would be suitable for that day's weather and temperature.

Building Weather Wear

This was the first solo app I built after learning the basics of Swift and common design principles from building smaller sandbox projects. Throughout this project, I challenged myself to dig deeper into how to structure my swift files, how to handle optionals, and identify possible memory leaks. I also researched how to build custom buttons that behave like radio buttons and checkboxes for my custom form. Overall I succeeded in creating my vision for this app and applied my self-learning to this solo project.

Highlights

  • Weather data received from OpenWeatherAPI using URLSession, JSONDecoder and mapped to a custom Weather Model.
  • The user chooses a location through the use MapKit and CoreLocation and the coordinates are used in the API call to get the weather data.
  • The user adds their clothing using a form made with a table view controller, custom buttons(radio and checkboxes), and input fields which are saved to core data as a clothing item.
  • Saved clothing items are shown in a collection view and searchable through the details the user inputted via the search controller.
  • All clothing items are saved in core data and presented to the user based on the day's temperature.
