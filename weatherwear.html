Weather Wear

What is Weather Wear?

Weather Wear is a weather app that will give you suggestions on what to wear based on the items of clothing you catalog in the app. The app works by letting you select your location by dropping a pin on the map and getting the 7-day outlook for that location. You can then select a day of the week and see what clothing would be suitable for that day's weather and temperature.

Feature Highlights

  • A Weather view controller shows the 7 day forecast using weather data received from OpenWeatherAPI. Creating a URLSession and JSONDecoder the data response is mapped to a weather model and used to populate the table cells.

  • A Map view controller allows the user to choose locations by dropping red pins on the map. Using MapKit and CoreLocation the coordinates of the pins are saved in UserDefaults and used in the API call to get the weather data.

  • The Clothing suggestion view controller uses a table view with a horizontal collection view for its table cells to display pictures of clothing items. The items shown are specific to the temperature for the day and are filtered based on the season category.

  • A CRUD feature that lets a user catalog clothing items in the app. The user adds their clothing using a form made with a table view controller, custom buttons(radio and checkboxes), and input fields which are saved to core data as a clothing item.

  • Saved clothing items are shown in a collection view and searchable through the details the user inputted via the search controller. Items can also be deleted through a multi-select and updated by populating the same form with prior details.
